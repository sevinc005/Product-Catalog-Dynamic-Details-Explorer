# Product-Catalog-Dynamic-Details-Explorer
🛒 Modern E-Commerce Catalog (React)
Bu layihə, DummyJSON API istifadə edərək hazırlanmış, real vaxt rejimində məlumatların idarə edilməsi, çoxmeyarlı filtrləmə və dinamik marşrutlaşdırma funksiyalarına malik müasir e-ticarət kataloqudur.

🚀 Texnologiyalar
React 18 - UI komponentləri üçün.

TanStack Query (React Query) - Asinxron məlumatların idarə edilməsi və keşlənməsi üçün.

Axios - REST API sorğuları üçün.

React Router DOM - Səhifələrarası keçid (Dynamic Routing) üçün.

Tailwind CSS - Modern və adaptiv (responsive) dizayn üçün.

✨ Əsas Funksiyalar
Dinamik Məhsul Siyahısı: API-dan gələn məlumatların asinxron şəkildə yüklənməsi və vizuallaşdırılması.

Çoxmeyarlı Filtrləmə: İstifadəçilər eyni anda həm məhsul adına (search), həm də kateqoriyaya (category) görə axtarış edə bilərlər.

Məhsul Təfərrüat Səhifəsi: Hər bir məhsul üçün id əsaslı xüsusi səhifə (Dynamic Detail Page).

Səhvlərin İdarə Edilməsi: Yüklənmə (Loading state) və xəta (Error handling) hallarının istifadəçi təcrübəsinə (UX) uyğun idarə olunması.

Adaptiv Dizayn: Mobil, planşet və desktop ekranları üçün tam uyğunlaşdırılmış (Responsive) interfeys.
